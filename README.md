# celebrities Data Management System

=================

In this project, I have created a celebrities Data Management System web App using React JS, Javascirpt, CSS, Bootstrap, HTML and css.

## To start a project

> npm install i

> npm start

## To access a project

Open a brower and type below:

> http://localhost:3000/
